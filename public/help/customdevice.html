JSON Syntax

This page can be used to modify the JSON syntax of the IoT messages. Any valid JSON will be accepted. Variables can be made by surrounding the variable name in double curly braces {{like_so}}.

You can edit the properties of a variable by double clicking it.

Variable Properties

There are a few different variable types that can be defined. The default is 'prompt'. You will be able to supply the value for this variable on the new device page.

The second type is 'Random number'. This variable type will randomly generate a value between min and max each time a message is sent.

The third type is 'javascript'. This type takes a javascript function called 'value'. Whatever value this javascirpt returns will be used as the JSON variable.
